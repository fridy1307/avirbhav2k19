PMJKS
(PUBG MOBILE JK SERIES)



Rules:

  1. In order to participate in this tournament, a Squad must consist of at least four players. All the players must be from JK.
  2. Only 3 teams at max from a house are allowed to participate.
  3. All players must use their Mobile Phones to play the game and all players must install the newest version of the game in order to participate in PUBGM JK Series.
  4. Squads are responsible for their own technical issues (hardware/internet). Matches will not be rescheduled because of technical issues and matches will continue to be played nevertheless. If the warm-up time in starting island is exceeded, the game goes on, even if the issue is not resolved.
  5. If any player, particular player is known to be using emulators or any other kind of 3rd party tools/hacks which allows players to play the game in PC, the whole squad will get disqualified from the tournament.
  6. In case a player gets disconnected from a match, the Squad is allowed to continue to play the match at a disadvantage.
